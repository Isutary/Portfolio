Library

The Library is a backend for an application that might be used for website about books and authors. It was created in ASP.net core and it follows RESTful principles. After gaining knowledge about ASP.net and RESTful, I wanted to see if I could create an application that would be as close to production level as possible. In the end, I created CRUD operations, authorization, and authentication for it in a RESTful manner.

You start off by running applications. Then you use login endpoint to generate JWT that will be used for your authentication and authorization. Depending on the user that you use to login you will have access to different CRUD operations. You have endpoints for CRUD operations with books, authors, roles, and users. You also have the ability to reset a password and register new users.
